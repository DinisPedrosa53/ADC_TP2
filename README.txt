Como correr o projeto:
1. Entrar no main.py;
2. Compilar o código;
3. Fazer login/criar utilizador;
4. Trabalhar com o menu;

Como construir e abrir a documentação a partir do terminal:
1. Criar um terminal python;
2. Usar o comando cd docs (Se estiver na pasta ..\ADCTP2);
3. Correr o comando python -m sphinx -b html . _build/html (Isto ativa o builder de docs do sphinx);
4. Para abrir a documentação correr o comando start _build/html/index.html;
